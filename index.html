Agendar Citas

Plataforma SaaS multi-marca para agendar citas.

Para ver Home, Agenda, Blog y el resto de pantallas debes usar el servidor de desarrollo Vite. Esta pagina (Apache/XAMPP) no carga React.

Abrir app en localhost:5173

Zonas de la aplicación (desarrollo)

  • App pública (clientes): http://localhost:5173
  • Demo marca: /m/luna-nails
  • Panel admin de marca: http://localhost:5173/admin/
  • API Node: http://localhost:3001/api/estado

El panel de operaciones internas (/plataforma) no se expone a administradores de marca.

Antes de abrir el navegador

cd backend && npm run dev
cd frontend && npm run dev

Solo necesitas MySQL de XAMPP para la base de datos. Apache no sirve esta app.
